# build-a-static-site
The #1 job of a static site generator is to convert Markdown into HTML.

Instead of writing a verbose HTML list:
<ul>
  <li>I really</li>
  <li>hate writing</li>
  <li>in raw html</li>
</ul>

We can write this in Markdown:
- I really
- hate writing
- in raw html

Markdown Cheat sheet 
https://www.markdownguide.org/cheat-sheet/
